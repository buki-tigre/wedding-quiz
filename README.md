# 모바일 청첩장 웹사이트

모바일 환경에 최적화된 현대적인 청첩장 웹사이트입니다.

## 주요 기능

- 모바일 최적화 레이아웃
- 신랑/신부 정보 표시
- 위치 및 교통편 안내
- 인터랙티브 퀴즈 기능
- 축하 메시지 등록
- 연락처 공유 기능

## 기술 스택

- HTML5
- CSS3 (애니메이션, 그라데이션 효과)
- 바닐라 JavaScript
- 반응형 디자인

## 실행 방법

이 웹사이트를 실행하려면 브라우저에서 `index.html` 파일을 열기만 하면 됩니다.

## 사용자 정의

커플 정보, 날짜, 장소, 사진 등을 사용자의 필요에 맞게 변경할 수 있습니다. `index.html` 파일에서 관련 텍스트를 수정하고, 이미지 URL을 자신의 이미지로 변경하세요.

## 개발자 정보

이 웹사이트는 Claude 3.7 Sonnet AI에 의해 제작되었습니다.
